# Minesweeper

A simple clone of the minesweeper game with 9x9 grid and 10 mines, written in python.

## Prerequisites

1. Python 2.7
1. Pygame

## Installation

```
pip install pygame
```
to install pygame

```
git clone https://github.com/khansubhan95/minesweeper
```

Enter into the cloned repo

```
python minesweeper.py
```
